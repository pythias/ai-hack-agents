# Security policy

This repository is a **deliberate defensive-testing harness**. It contains synthetic adversarial text (for example under `issues/` and in GitHub templates) used to evaluate agent behavior. That content is **not** an invitation to follow those instructions against real systems.

- Do **not** place real API keys, tokens, or personal secrets in this repo. Use only the fake samples under `fixtures/`.
- If you find **unintentional** exposure of real credentials or a flaw in the evaluation harness itself, please open a private security advisory on GitHub or contact the repository maintainers.
